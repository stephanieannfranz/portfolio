quinn nails

branding // logo design // website design

Quinn Nails is a local nail salon that I've gone to several times. I noticed that the salon didn't have any online presence, so I approached the owner (Quinn) and asked her if she needed a website. She was interested but wasn't sure exactly how to start the process. I decided to start by focusing on the various design elements.

Now that I've completed the design, I'll work with Quinn through the website creation process, from purchasing a domain name to adding content in WordPress.

logo creation

Quinn Nails doesn't have much of an online presence, and the only branded materials is the menu of options given to you when you visit the salon. That's why I wanted to start of with something basic: a new, simplified logo that's ideal for using on the web.

I was inspired by drops of nail polish, so I used Adobe Illustrator to create a logo that looks like a paint drop. I also created a couple of variations, which could be used other branded items.

style tiles

After creating the logo and gathering inspiration, I decided to present my design ideas using Style Tiles. The idea behind style tiles is that you create the personality and components of a website, without committing to a full-blown website design.

I used adjectives given to me by Quinn to come up with colors and the personality of the website, then I created the elements based on those. The colors, text styles, and button style will be used across the website.

prototype

I then decided to create a prototype using Adobe XD. This is a relatively new software, so I thought it would be fun to play around with the prototyping feature. I chose to create the mobile site, since I tend to approach design with a "mobile first" philosophy. I wanted it to be easy to use with all the information a customer might want immediately accessible.

The Adobe XD prototype is embedded to the left (or below). You can scroll and click through like you would an actual mobile site.
